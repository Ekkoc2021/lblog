# LBlog 首页接口文档

基础路径: `/api/v1`

通用响应格式:
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

---

## 1. 获取文章列表

**GET** `/posts`

首页核心接口，支持分页、排序、筛选。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认1 |
| pageSize | int | 否 | 每页条数，默认10 |
| sort | string | 否 | 排序方式: `recommend`(推荐，默认) / `newest`(最新) / `hot`(最热) |
| categoryId | long | 否 | 按分类筛选 |
| tagId | long | 否 | 按标签筛选 |
| seriesId | long | 否 | 按专栏筛选 |
| keyword | string | 否 | 搜索关键词（标题+摘要模糊匹配） |

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "title": "Spring Boot 3.0 新特性全面解析",
        "slug": "spring-boot-3-features",
        "excerpt": "本文全面介绍Spring Boot 3.0带来的新特性...",
        "featuredImage": null,
        "status": 1,
        "authorId": 1,
        "categoryId": 1,
        "publishedAt": "2026-04-24T10:00:00",
        "createdAt": "2026-04-24T10:00:00",
        "updatedAt": "2026-04-24T10:00:00",
        "author": {
          "id": 1,
          "nickname": "Ekko",
          "avatar": null
        },
        "category": {
          "id": 1,
          "name": "后端",
          "slug": "backend"
        },
        "tags": [
          { "id": 1, "name": "Java", "slug": "java" },
          { "id": 2, "name": "Spring Boot", "slug": "spring-boot" }
        ],
        "viewCount": 1256,
        "likeCount": 89,
        "commentCount": 23
      }
    ],
    "total": 42,
    "page": 1,
    "pageSize": 10
  }
}
```

### 说明

- `sort=recommend`: 按综合权重排序（考虑发布时间、浏览量、点赞数等）
- `sort=newest`: 按 `published_at` 降序
- `sort=hot`: 按 `view_count` 降序
- 只返回 `status=1`（已发布）的文章
- `viewCount`、`likeCount`、`commentCount` 为冗余字段，后续需在对应表中添加

## 2. 获取分类列表

**GET** `/categories`

导航栏及筛选使用。

### 请求参数

无

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "后端",
      "slug": "backend",
      "parentId": null,
      "description": "后端开发",
      "sortOrder": 0,
      "postCount": 42
    }
  ]
}
```

### 说明

- `postCount` 为该分类下已发布文章的数量，需后端统计
- 只返回顶级分类（`parent_id IS NULL`），子分类后续按需扩展

---

## 3. 获取标签列表

**GET** `/tags`

侧边栏热门标签展示。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| limit | int | 否 | 返回数量，默认20 |

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "Java",
      "slug": "java",
      "postCount": 25
    }
  ]
}
```

### 说明

- `postCount` 为该标签关联的已发布文章数
- 默认按 `postCount` 降序排列

---

## 4. 获取专栏列表

**GET** `/series`

侧边栏专栏推荐展示。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| limit | int | 否 | 返回数量，默认5 |
| categoryId | long | 否 | 按分类筛选 |

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "id": 1,
      "title": "Spring Boot 实战",
      "slug": "spring-boot-in-action",
      "description": "从零开始学习Spring Boot",
      "coverImageUrl": null,
      "categoryId": 1,
      "isCompleted": 0,
      "sortOrder": 0,
      "postCount": 12
    }
  ]
}
```

### 说明

- `postCount` 为该专栏关联的已发布文章数，通过 `series_posts` 表统计
- `isCompleted`: 0-连载中, 1-已完结

---

## 5. 获取热门文章

**GET** `/posts/hot`

侧边栏热门文章排行。

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| limit | int | 否 | 返回数量，默认5 |

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "id": 3,
      "title": "MySQL 索引优化实战：从慢查询到毫秒级响应",
      "slug": "mysql-index-optimization",
      "viewCount": 2341
    }
  ]
}
```

### 说明

- 按 `view_count` 降序排列，只返回精简字段

## 四、验证标准

完成后通过以下测试：

- [x] `POST /api/v1/auth/login` — 正确账号返回双 token + 用户信息
- [x] `POST /api/v1/auth/login` — 错误密码返回 401
- [x] `GET /api/v1/auth/me` — 携带有效 access_token 返回用户信息
- [x] `GET /api/v1/auth/me` — 无 token 返回 401
- [x] `GET /api/v1/home/**` — 无需 token 可访问
- [x] `/api/v1/admin/**` — 无 token 返回 401
- [x] `POST /api/v1/auth/logout` — 登出后 token 失效
- [x] 现有用户的旧密码（明文）仍可登录（DelegatingPasswordEncoder 兜底）

## 边界覆盖情况

| 边界 | 覆盖方式 | 结论 |
|------|----------|------|
| 空结果集 | categoryId不存在(如999) → 返回空list, total=0 | ✅ 隐式覆盖 |
| 超大分页值 | pageSize=100 → 正常返回(max=100) | ✅ 注解控制 |
| 非法sort值 | sort=invalid → 回退recommend | ✅ |
| 草稿过滤 | post(id=12, status=0) → 不在结果中 | ✅ |
| 软删除 | is_delelte=0条件 → 所有数据均满足 | ✅ 逻辑保证 |
| 参数校验 | page=0, limit=0 → 400错误 | ✅ |
| 多标签 | post(id=1)有2个标签 → tags数组含2项 | ✅ |
| 无标签文章 | 所有文章均有标签 | ✅ 测试数据覆盖 |
| NULL值字段 | featuredImage, avatar等 → 返回null | ✅ |

全部 **20 个测试用例** 通过，接口行为符合 `2首页接口文档.md` 定义。无需新增测试数据。
